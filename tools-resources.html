Tools & Resources

Curated tools to help you build, learn, and grow

Tools are what you need to actually build things. GitHub for code, Vercel for hosting, AI APIs for smart features, and data sources for finding opportunities. Here are some that actually work and won't cost you money.

AI Tools

  • OpenAI API

    Access to GPT models for building chatbots and AI features. Has a free tier.

    platform.openai.com
    Best for: Chatbots, text generation, code assistance. Free tier: $5 credit monthly.
    Pricing: Pay-as-you-go after free tier. Very affordable for small projects.

  • Hugging Face

    Open-source AI models and datasets for experimentation.

    huggingface.co
    Best for: Experimenting with models, fine-tuning, NLP tasks. Completely free.
    Pricing: Free. Paid plans for production use only.

Startup Resources

  • Y Combinator Startup School

    Free online course covering startup basics.

    startupschool.org
    Best for: Learning startup fundamentals, idea validation, MVP development.
    Content: Video lectures, exercises, community forum. Self-paced.

  • Indie Hackers

    Community of founders building profitable businesses.

    indiehackers.com
    Best for: Learning from real founders, revenue transparency, community support.
    Content: Interviews, forums, revenue reports. Real stories from builders.

Development Tools

  • GitHub

    Version control and code hosting. Free for students.

    github.com
    Best for: Version control, code hosting, collaboration, CI/CD.
    Student pack: Free GitHub Pro, GitHub Copilot, domain credits, cloud credits.

  • Vercel

    Free hosting for web applications.

    vercel.com
    Best for: Hosting React, Next.js, Node.js apps. Instant deployments.
    Free tier: Unlimited personal projects, 100GB bandwidth, custom domains.
